# rb-swatch - Internet Time converter

# EXAMPLE

```
$ swatch
@150.00
$ swatch
@150.01
$ swatch
@150.02
...
```

# HOMEPAGE

https://github.com/mcandre/rb-swatch

# RUBYGEMS

https://rubygems.org/gems/rb-swatch

# REQUIREMENTS

* [Ruby](https://www.ruby-lang.org/) 2.0+

# INSTALL

Install via [RubyGems](http://rubygems.org/):

```
$ gem install rb-swatch
```
